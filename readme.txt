== Description ==

Clean Researcher is a minimal, article-first WordPress theme for long-form writing.

It is designed to keep the reading experience clean while still giving readers quick navigation through a built-in table of contents (TOC).

Main features:

* Clean, centered reading layout for posts and pages.
* Automatic table of contents generated from your headings.
* Mobile TOC drawer for small screens.
* Customizer controls for title and body fonts.
* Adjustable content width from the Customizer.
* Optional TOC on pages.
* TOC depth setting (H2 only by default, with H3/H4 options).

== Installation ==

1. Upload the `clean-researcher` folder to `/wp-content/themes/`.
2. Go to `Appearance > Themes` in WordPress admin.
3. Activate **Clean Researcher**.

== Recommended Setup ==

1. Create a menu at `Appearance > Menus` and assign it to the Primary Menu location.
2. Go to `Appearance > Customize > Clean Researcher Theme`.
3. Set your preferred:
	 * Title font
	 * Body font
	 * Content width
	 * TOC behavior on pages
	 * TOC maximum depth

For best TOC results, structure posts with clear heading levels:

* Use `H2` for major sections.
* Use `H3` and `H4` only for nested sections.

== Frequently Asked Questions ==

= Why is the table of contents not showing? =

The TOC is automatically hidden if there are fewer than 2 headings in the content.

Add at least two section headings (for example two `H2` headings), then refresh the page.

= How do I change which headings are included in the TOC? =

Go to `Appearance > Customize > Clean Researcher Theme > TOC maximum depth`.

Options:

* H2 only (default)
* H2 + H3
* H2 + H3 + H4

= Can I show TOC on pages as well as posts? =

Yes. Enable it in `Appearance > Customize > Clean Researcher Theme`.

= Does this theme support the block editor? =

Yes. The theme supports standard WordPress content editing and styles article content using the built-in prose styles.
